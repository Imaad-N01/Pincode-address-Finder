# Pincode Address Finder

A simple web application that allows users to enter an Indian **Pincode** and get the corresponding **Post Office, District, State, and Country details** instantly.

This project demonstrates how to work with **APIs, HTML, CSS, and JavaScript** to build a useful real-world application.

---

## 🚀 Features

* Search address details using a **6-digit Indian Pincode**
* Displays:

  * Post Office Name
  * District
  * State
  * Country
* Input validation for numeric pincodes
* Clean and responsive user interface
* Floating label input design

---

## 🛠 Technologies Used

* **HTML5** – Structure of the webpage
* **CSS3** – Styling and layout
* **JavaScript** – API handling and validation
* **India Post Pincode API**

## ⚙️ How It Works

1. User enters a **6-digit pincode**.
2. JavaScript sends a request to the **Pincode API**.
3. The API returns location details.
4. The webpage displays:

   * Post Office
   * District
   * State
   * Country
